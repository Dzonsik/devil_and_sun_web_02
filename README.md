# Devil and Sun tattoo studio Web

Toto je webová aplikace pro tetovací studio Devil and Sun tattoo.

## Funkce
- Úvodní stránka
- Portfolio stránka
- Proces stránka
- O mně stránka
- Další tvorba stránka
- Registrace schůzky stránka + backend
- Integrace databáze
